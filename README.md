# Chatbot de Gestión de Vacaciones
# TPI---Trabajo-Integrador-Chatbot-Organizaci-n-Empresarial---Benitez-Nayla
Trabajo Práctico Integrador - Organización Empresarial
BENITEZ NAYLA

## Descripción

Este proyecto implementa un chatbot que automatiza el proceso de solicitud de vacaciones dentro de una organización.

El sistema consulta una base de datos simulada mediante un archivo CSV y verifica si el empleado posee días suficientes para aprobar la solicitud.

## Tecnologías

- Python
- CSV
- BPMN 2.0
- Draw.io

## Funcionalidades

- Validación de legajo
- Consulta de días disponibles
- Aprobación automática
- Rechazo automático
- Gestión mediante máquina de estados
